Jolly Good Toolbelt¶

A collection of tools shared by jolly good developers.

Libraries¶

Several libraries have been created that can be installed via pip:

Library Name

Description

jgt_common

A library for helper functions across Jolly Good Toolbelt designed for larger consumption

jgt_tools

A collection of tools for commmon package scripts

jiratools

CLI Helpers for quick JIRA actions

jolly_github_toolbelt

A collection of tools around Git and GitHub to make developers’ lives easier.

layered-config

A tool for managing layered config files!

sphinx-gherkindoc

A tool to convert Gherkin into Sphinx documentation

tableread

A library for converting reStructredText tables into Python objects

In addition, there is a library for TypeScript to assist in UI visual regression testing: testcafe-visual-regression.
